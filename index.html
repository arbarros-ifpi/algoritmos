Unidade 1 — Introdução à lógica e algoritmos

Conceito de algoritmo

Algoritmo é uma sequência lógica de passos para resolver um problema.

Exemplo simples

algoritmo "soma"

var
a, b, soma: inteiro

inicio

leia(a)
leia(b)

soma <- a + b

escreva(soma)

fimalgoritmo

Exercícios

  1. Faça um algoritmo que leia dois números e mostre a soma.
  2. Faça um algoritmo que calcule a média entre duas notas.
  3. Desenvolva um algoritmo que calcule a velocidade média.

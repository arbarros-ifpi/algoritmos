# Unidade 1 — Introdução à lógica e algoritmos

## Conceito de algoritmo

Algoritmo é uma sequência lógica de passos para resolver um problema.

---

## Exemplo simples

```portugol
algoritmo "soma"

var
a, b, soma: inteiro

inicio

leia(a)
leia(b)

soma <- a + b

escreva(soma)

fimalgoritmo
```
